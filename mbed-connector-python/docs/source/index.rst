mdc-api-python docs
==========================

Welcome to the documentation for the mdc-api-python package. This documentation goes over the API and examples for using the mdc-api python module. 
This python package is an interface to the mbed Device Connector REST API. The mdc-api-python is meant to make it easy to control mbed client devices through mbed Device Connector. 
Please check out the API and Examples included in this doc.

Happy Hacking!
